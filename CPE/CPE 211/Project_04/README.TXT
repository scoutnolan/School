The input files in this zip file are entered when prompted for
by the program.

Verify that your program correctly analyzes the contents of 
all the input files.

The input files contain the following information (4 lines in the input file):

First line that is divided into two parts by a single unique character
the unique character used to parse(split) the first line
Second line that is divided into 3 parts by two occurences of a unique character
the unique character that parses(splits) the second line

The unique character used to divide the lines can be different between the two lines.
The unique character can occur anywhere in the lines that are to be parsed(split)

Your program should run for all input files.
